# Stimulus Components

## Info

### ControlSelect

```javascript
import { useStimulus } from '../../composables/stimulus.js'
import { ControlSelect } from '@newlogic-digital/stimulus-components'

useStimulus.register('x-control-select', ControlSelect)
```
